 # FlavorHub
App is a smart and user-friendly dining companion designed to help people discover top restaurants nearby effortlessly. Using the Foursquare Places API (or another free location service), the app combines live restaurant data, local storage, and a personalized experience to create a smooth and engaging food discovery journey.

## Core Features

### Authentication
- Login and Register screens
- New users can create an account
- Existing users can log in easily
- User data is stored locally

### Find Nearby Restaurants
- Discover restaurants near your location
- Uses Places API for live restaurant data
- Supports pagination (20 restaurants per load)
- Local caching for faster performance

### Reviews
- Add and read restaurant reviews
- Reviews are saved locally
- Revisit your saved feedback anytime

### Hide Restaurants
- Thumbs down restaurants you dislike
- Hidden restaurants are filtered automatically
- Personalized restaurant experience

## How the App Works

1. Register or log in
2. Fetch nearby restaurants using the Foursquare API
3. Save restaurant data locally
4. Add and view reviews
5. Hide unwanted restaurants
